Overview

Allen's Kitchen is a full-featured online food ordering and restaurant management platform — the flagship project of my portfolio. Built from scratch, it handles everything from customer browsing and ordering to kitchen management and admin reporting.

The system targets small to medium restaurants that need a robust, affordable digital presence without relying on third-party delivery platforms that take large commissions.

Core Features

  • Customer-facing menu with real-time item availability, categories, and search
  • Full ordering flow — cart, checkout, order tracking with live status updates
  • Admin panel for menu management, order processing, and revenue reporting
  • Kitchen display system showing incoming orders in priority order
  • User authentication with role-based access (customer, staff, admin)
  • Mobile-responsive design built with Tailwind CSS utility classes
  • MySQL database with relational schema for orders, items, users, and inventory
// challenge

The trickiest part was syncing order status in real time between the customer view and kitchen display without WebSockets — solved using lightweight polling with PHP endpoints and optimistic UI updates on the frontend.

// what i learned

This project pushed me deep into database normalisation, session management, and building a full authentication flow from scratch. It also taught me how to structure a multi-role application where the same backend serves completely different UI needs.

Overview

A fully-featured admin dashboard built with TypeScript — my first major project using static typing in a real application. The dashboard provides a clean management interface with data tables, analytics views, and user management.

The focus was on writing maintainable, type-safe code and building reusable component patterns that could be dropped into any backend system.

Core Features

  • Sortable and filterable data tables with pagination
  • Analytics charts showing key metrics with visual breakdowns
  • User management interface with role assignment
  • Responsive sidebar navigation with collapsible sections
  • Dark/light mode toggle with persistent preference
  • Fully typed props and API response interfaces with TypeScript
// challenge

Learning TypeScript's type system while building a real project simultaneously was a steep curve. Generic types for reusable table components and correctly typing API responses took multiple iterations to get right.

// what i learned

TypeScript forced me to think about data shapes before writing logic — a discipline that made the codebase significantly easier to debug. I also got solid practice in component architecture and prop drilling vs context patterns in React.
